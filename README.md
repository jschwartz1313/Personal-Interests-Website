# My Interests

A personal interests tracker for movies, TV shows, sports, fandoms, and professional topics.

## Features

- Interactive expandable cards for different interests
- Star rating system for each interest
- Personal notes for each card
- Media tracking with checkboxes and ratings
- Data export/import functionality
- Auto-backup system
- Filter interests by rated/unrated/with notes

## Live Site

Visit the live site at: https://[your-username].github.io/interests

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- localStorage & IndexedDB for data persistence

## Usage

Simply open the site and start exploring your interests! Click on any card to expand it, rate it with stars, and add personal notes.

## Data Storage

All your ratings and notes are stored locally in your browser. Use the export feature to download a backup of your data.
