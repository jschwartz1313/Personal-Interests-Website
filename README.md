## Features

### Main Page (index.html)
- Interactive expandable cards for different interests
- 5-star rating system for each interest
- Personal notes for each card
- Media tracking with checkboxes and ratings
- Quick links to external resources
- Data export/import functionality
- Auto-backup system
- Filter interests by rated/unrated/with notes

### Stats Dashboard (stats.html)
- Overview cards showing total interests, ratings, and notes
- Category breakdown visualization
- Top-rated interests leaderboard
- Rating distribution bar charts
- Recent activity timeline
- Fun facts with gradient cards

### Gallery (gallery.html)
- Curated image collection of your interests
- Filter by category (fandoms, movies, sports, tech)
- Beautiful lightbox for viewing images
- Hover effects showing descriptions
- Shows your ratings on each image

### Timeline (timeline.html)
- Visual journey showing interest evolution
- Major milestones tracker
- Past, present, and future sections
- Animated timeline with icons
- Personal growth documentation

### Add Interest (add.html)
- Interactive form with live preview
- Add custom sections and descriptions
- Track related media items
- Add quick links to resources
- Saves custom interests to localStorage

## Tech Stack

- HTML5
- CSS3 (Grid, Flexbox, Animations, Gradients)
- Vanilla JavaScript (ES6+)
- localStorage & IndexedDB for data persistence
- Unsplash API for placeholder images

## Usage

1. **Explore Interests**: Click cards to expand and see details
2. **Rate Everything**: Use 5-star system to rate your interests
3. **Add Notes**: Write personal thoughts and memories
4. **Track Media**: Check off books, movies, shows you've experienced
5. **View Stats**: See analytics and insights on the Stats page
6. **Browse Gallery**: Enjoy visual representation of interests
7. **Review Timeline**: Track your journey through time
8. **Add Custom**: Create your own interests with the Add page
9. **Export Data**: Download backup anytime via data management panel

## Data Storage

All your ratings and notes are stored locally in your browser. Use the export feature to download a backup of your data.

## Getting Started

Simply open `index.html` in your browser or use Live Server in VSCode for the best experience!
